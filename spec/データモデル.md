# ER図

以下の情報を元に、ent/schema内のファイルを作成する

```mermaid
erDiagram
    USERS {
        uuid id
        string name
    }

    TENANTS {
        uuid id
        string owner_id
        string name
    }

    PRODUCTS {
        uuid id
        string tenant_id
        string category_id
        string name
        number price
        json properties
    }

    CATEGORIES {
        uuid id
        string name
    }

    USERS ||--o{ TENANTS : owns
    TENANTS ||--o{ PRODUCTS : has
    CATEGORIES ||--o{ PRODUCTS : categorizes

```
